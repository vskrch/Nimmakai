Welcome to Nimmakai Pro

Nimmakai is a state-of-the-art OpenRouter-style proxy. To get started:

  1. Add Providers: Head to the Providers tab and add your API keys.
  2. Configure Routing: The system automatically builds performance ladders. You can pin specific models in the Routing tab.
  3. Test: Jump into the Playground to stream completions.
  4. Trace: Monitor your token usage in the Tracing tab.
